# Fontes locais

Coloque aqui os arquivos da fonte **Telegraph** para uso via `@font-face`.

Arquivos esperados (pelo menos um dos formatos):

- `Telegraph-Regular.woff2` e/ou `Telegraph-Regular.woff` (peso 400)
- `Telegraph-Bold.woff2` e/ou `Telegraph-Bold.woff` (peso 700)

Se os arquivos tiverem outros nomes, edite `src/styles/globals.css` para apontar para os nomes corretos.

Sem esses arquivos, o site usará a fonte de fallback (system-ui).
